3. Pilih Threshold:#

Ambil Threshold = 155 cm

Maka dibagi menjadi:

  • Group Left (≤ 155): No 3, 2, 4, 11, 1, 14, 12, 5, 9 → 9 data

  • Group Right (> 155): No 6, 13, 8, 15, 7, 10 → 6 data

4. Hitung Entropy Awal#

\[ p_1 = \frac{8}{15}, \quad p_2 = \frac{7}{15} \]
\[ Entropy(D) = -\frac{8}{15} \log_2 \frac{8}{15} - \frac{7}{15} \log_2 \frac{7}{15} \]

Hitung:

  • \(\frac{8}{15} \approx 0.533\)

  • \(\frac{7}{15} \approx 0.467\)

Maka:

\[ Entropy(D) ≈ -0.533 \cdot \log_2(0.533) - 0.467 \cdot \log_2(0.467) ≈ 0.996 \]

5. Entropy Setelah Split (Threshold 155 cm)#

Group Left (≤ 155 cm)#

Data: No 3, 2, 4, 11, 1, 14, 12, 5, 9 → 9 data

  • Gorilla gorilla = 8

  • Gorilla beringei = 1

\[ Entropy(left) = -\frac{8}{9} \log_2 \frac{8}{9} - \frac{1}{9} \log_2 \frac{1}{9} \]

  • \(\frac{8}{9} ≈ 0.889\), \(\frac{1}{9} ≈ 0.111\)

\[ Entropy(left) ≈ -0.889 \cdot \log_2(0.889) - 0.111 \cdot \log_2(0.111) ≈ 0.503 \]

Group Right (> 155 cm)#

Data: No 6, 13, 8, 15, 7, 10 → 6 data

  • Gorilla gorilla = 0

  • Gorilla beringei = 6

\[ Entropy(right) = -1 \cdot \log_2(1) = 0 \]

6. Information Gain#

\[ IG = Entropy(D) - \left( \frac{9}{15} \cdot Entropy(left) + \frac{6}{15} \cdot Entropy(right) \right) \]
\[ IG = 0.996 - \left( \frac{9}{15} \cdot 0.503 + \frac{6}{15} \cdot 0 \right) = 0.996 - 0.3018 = \boxed{0.694} \]

7. Gini Index#

Gini Awal:#

\[ Gini(D) = 1 - \left( \frac{8}{15} \right)^2 - \left( \frac{7}{15} \right)^2 ≈ 1 - 0.284 - 0.218 ≈ 0.498 \]

Gini Left:#

  • 8 Gorilla gorilla, 1 Gorilla beringei

\[ Gini(left) = 1 - \left( \frac{8}{9} \right)^2 - \left( \frac{1}{9} \right)^2 = 1 - 0.790 - 0.012 ≈ 0.198 \]

Gini Right:#

  • 6 Gorilla beringei → Pure class

\[ Gini(right) = 0 \]

Gini Split:#

\[ Gini_{split} = \frac{9}{15} \cdot 0.198 + \frac{6}{15} \cdot 0 = \boxed{0.1188} \]

2. Ambil threshold: 100 kg#

Karena ada perubahan dominasi kelas dari Gorilla gorilla ke Gorilla beringei setelah 100 kg.

Maka:#

  • Group Left (≤ 100 kg): No 5, 2, 3, 6, 10, 1, 11, 4, 13, 7 → 10 data

  • Group Right (> 100 kg): No 12, 15, 8, 14, 9 → 5 data

3. Komposisi kelas:#

Sebelum split:#

  • Gorilla gorilla: 8

  • Gorilla beringei: 7

\[ Entropy(D) = -\frac{8}{15} \log_2 \frac{8}{15} - \frac{7}{15} \log_2 \frac{7}{15} \approx 0.996 \]

4. Entropy Setelah Split#

Group Left (≤ 100):#

  • Gorilla gorilla: 6 (No: 5, 2, 3, 1, 11, 4)

  • Gorilla beringei: 4 (No: 6, 10, 13, 7)

\[ Entropy(left) = -\frac{6}{10} \log_2 \frac{6}{10} - \frac{4}{10} \log_2 \frac{4}{10} \]

  • \(\frac{6}{10} = 0.6, \quad \log_2(0.6) ≈ -0.737\)

  • \(\frac{4}{10} = 0.4, \quad \log_2(0.4) ≈ -1.322\)

\[ Entropy(left) ≈ -0.6 \cdot (-0.737) - 0.4 \cdot (-1.322) ≈ 0.442 + 0.529 = 0.971 \]

Group Right (> 100):#

  • Gorilla gorilla: 2 (No: 12, 14)

  • Gorilla beringei: 3 (No: 15, 8, 9)

\[ Entropy(right) = -\frac{2}{5} \log_2 \frac{2}{5} - \frac{3}{5} \log_2 \frac{3}{5} \]

  • \(\frac{2}{5} = 0.4, \quad \log_2(0.4) ≈ -1.322\)

  • \(\frac{3}{5} = 0.6, \quad \log_2(0.6) ≈ -0.737\)

\[ Entropy(right) ≈ -0.4 \cdot (-1.322) - 0.6 \cdot (-0.737) ≈ 0.529 + 0.442 = 0.971 \]

5. Information Gain#

\[ IG = Entropy(D) - \left( \frac{10}{15} \cdot 0.971 + \frac{5}{15} \cdot 0.971 \right) = 0.996 - 0.971 = \boxed{0.025} \]

6. Gini Index#

Gini Awal:#

\[ Gini(D) = 1 - \left(\frac{8}{15}\right)^2 - \left(\frac{7}{15}\right)^2 ≈ 1 - 0.284 - 0.218 = 0.498 \]

Gini Left:#

  • 6 Gorilla gorilla, 4 Gorilla beringei

\[ Gini(left) = 1 - (0.6)^2 - (0.4)^2 = 1 - 0.36 - 0.16 = 0.48 \]

Gini Right:#

  • 2 Gorilla gorilla, 3 Gorilla beringei

\[ Gini(right) = 1 - (0.4)^2 - (0.6)^2 = 1 - 0.16 - 0.36 = 0.48 \]

Gini Split:#

\[ Gini_{split} = \frac{10}{15} \cdot 0.48 + \frac{5}{15} \cdot 0.48 = \boxed{0.48} \]

Visualisasi Decision Tree#

Ini adalah code program untuk mencari decision tree sekaligus rules

import pandas as pd
import matplotlib.pyplot as plt
import numpy as np
from sklearn.tree import DecisionTreeClassifier, plot_tree, export_text
from sklearn.preprocessing import LabelEncoder

# === Data Gorilla ===
data = {
    "Panjang Tubuh (cm)": [150, 140, 135, 145, 155, 160, 170, 165, 155, 175, 148, 152, 162, 143, 168],
    "Berat Tubuh (kg)":   [89, 80, 85, 94, 80, 88, 100, 160, 180, 88, 90, 125, 98, 168, 140],
    "Kelas": [
        "Gorilla gorilla", "Gorilla gorilla", "Gorilla gorilla", "Gorilla gorilla", "Gorilla gorilla",
        "Gorilla beringei", "Gorilla beringei", "Gorilla beringei", "Gorilla beringei", "Gorilla beringei",
        "Gorilla gorilla", "Gorilla gorilla", "Gorilla beringei", "Gorilla gorilla", "Gorilla beringei"
    ]
}
df = pd.DataFrame(data)

# === Label Encoding ===
le = LabelEncoder()
df['Label'] = le.fit_transform(df['Kelas'])  # gorilla=0, beringei=1 (or vice versa)

# === Fitur dan Target ===
X = df[["Panjang Tubuh (cm)", "Berat Tubuh (kg)"]]
y = df["Label"]

# === Train Decision Tree ===
clf = DecisionTreeClassifier(criterion="gini", max_depth=3, random_state=42)
clf.fit(X, y)

# === 1. Tampilkan Pohon Keputusan ===
print("\n=== Pohon Keputusan (Decision Tree) ===")
tree_rules = export_text(clf, feature_names=list(X.columns))
print(tree_rules)

|   |--- class: 1
|--- Panjang Tubuh (cm) >  153.50
|   |--- Berat Tubuh (kg) <= 84.00
|   |   |--- class: 1
|   |--- Berat Tubuh (kg) >  84.00
|   |   |--- class: 0

Visualisasi Splitting#

# === Uji 2 Data Baru ===
uji_baru = pd.DataFrame({
    'Panjang Tubuh (cm)': [158, 142],
    'Berat Tubuh (kg)': [130, 87]
})

# Prediksi dan interpretasi label
prediksi = clf.predict(uji_baru)
hasil = le.inverse_transform(prediksi)

print("\n=== Prediksi 2 Data Baru ===")
for i, (p, b) in enumerate(zip(uji_baru['Panjang Tubuh (cm)'], uji_baru['Berat Tubuh (kg)'])):
    print(f"Data {i+1}: Panjang={p}, Berat={b} → Prediksi: {hasil[i]}")

# === Visualisasi Decision Boundary ===
print("\n=== Visualisasi Decision Boundary ===")
plot_colors = "ry"
plot_step = 1.0

x_min, x_max = X['Panjang Tubuh (cm)'].min() - 5, X['Panjang Tubuh (cm)'].max() + 5
y_min, y_max = X['Berat Tubuh (kg)'].min() - 10, X['Berat Tubuh (kg)'].max() + 10

xx, yy = np.meshgrid(np.arange(x_min, x_max, plot_step),
                     np.arange(y_min, y_max, plot_step))

grid = np.c_[xx.ravel(), yy.ravel()]
grid_df = pd.DataFrame(grid, columns=['Panjang Tubuh (cm)', 'Berat Tubuh (kg)'])
Z = clf.predict(grid_df)
Z = Z.reshape(xx.shape)

plt.figure(figsize=(8, 6))
plt.contourf(xx, yy, Z, cmap=plt.cm.RdYlBu, alpha=0.6)

# Plot data latih
for label, color in zip([0, 1], plot_colors):
    idx = np.where(y == label)
    plt.scatter(X.iloc[idx]['Panjang Tubuh (cm)'], X.iloc[idx]['Berat Tubuh (kg)'],
                c=color, edgecolor='k', label=le.inverse_transform([label])[0])

# Plot titik data uji
plt.scatter(uji_baru['Panjang Tubuh (cm)'], uji_baru['Berat Tubuh (kg)'],
            c='black', marker='x', s=100, label='Data Uji')

plt.xlabel("Panjang Tubuh (cm)")
plt.ylabel("Berat Tubuh (kg)")
plt.title("Decision Boundary (Gorilla gorilla vs Gorilla beringei)")
plt.legend()
plt.grid(True)
plt.tight_layout()
plt.show()

Data 2: Panjang=142, Berat=87 → Prediksi: Gorilla gorilla
